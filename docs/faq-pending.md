# FAQ utilisateurs - PENDING

## Besoin
Les utilisateurs (autonomes et coaches) ont besoin de comprendre
comment fonctionne l'app. Une FAQ accessible depuis l'app expliquant :
- Le fonctionnement pour les **autonomes** (ceux qui creent leurs
  propres programmes via AI, suivent leurs poids/photos/seances
  eux-memes)
- Le fonctionnement pour les **coaches** (gestion clients, creation
  templates, assignation, suivi progression, push update)
- Le fonctionnement pour les **clients invites** (recoivent un
  programme du coach, doivent suivre les seances, peuvent voir
  les modifications du coach)

## A definir avant l'implementation
- Format : page dediee /faq ou drawer accessible depuis settings ?
- Categories : par persona (autonome / coach / invite) ou par
  fonctionnalite (programmes / nutrition / suivi) ?
- Recherche ? Pas necessaire pour MVP.
- Maintenance : qui ecrit le contenu ? probablement Marco lui-meme
  ou Anthropic Claude pour la 1ere version.

## Impact / priorite
- Pas critique mais important pour l'onboarding des nouveaux users
- A faire avant le lancement payant ou apres premier feedback
- Reduit les questions support

## Effort estime
- 30 min : structure HTML/CSS de la page FAQ
- 2-3h : redaction du contenu (reflet du fonctionnement actuel)
- 30 min : navigation/lien depuis settings ou onboarding

Total : ~3-4h pour version v1

## Pieces necessaires pour le contenu
- Captures d'ecran annotes (autonome / coach / invite)
- Liste des fonctionnalites par persona
- Cas d'usage typiques (ex: "comment changer mon programme")
